# Mockup Prompt Templates

A curated collection of high-performance prompt templates to generate clean, professional UI/UX mockups from screenshots.

## Why These Prompts?

These templates turn rough screenshots into **pixel-perfect, production-quality mockups** with modern design polish while staying faithful to the original design.

## Available Templates

### Core Templates
- **[Basic Mockup Prompt](./mockup-from-screenshot.md)** — Best starting point for most use cases
- **[Enhanced Mockup Prompt](./mockup-from-screenshot-enhanced.md)** — Maximum quality & control (recommended)

### Specialized Templates
- **[Dark mode versions](./dark-mode-mockup.md)** - Use this to convert any screenshot into a premium dark mode version.**
- **[SaaS dashboard refinement](./saas-dashboard-mockup.md)** - Best for turning any interface into a polished SaaS-style dashboard. 

## How to Use

1. Go to any template file
2. Copy the full prompt
3. Attach your screenshot(s) at the end

(Optional) Add custom instructions like:
   - "Convert to dark mode"
   - "Make it a responsive desktop landing page"
   - "Use a glassmorphic / neumorphic style"

### Example Usage
```text
[ Paste the full template here ]

Additional instructions: 
Make this mobile app screen into a beautiful desktop web version with Tailwind-style clean aesthetics.

Here is the screenshot:
[attach image]
